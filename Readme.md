# Libreria de números complejos
#
### Las operaciones que podemos realizar con esta libreria son:
- Suma
- Producto
- Resta
- División
- Módulo
- Conjugado
- Conversión entre coordenadas polares y cartesianas
- Obtener la fase de un número complejo
